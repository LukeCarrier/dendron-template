## Summary

Process for upgrading CLI via `yarn` or `npm`.

## Steps

> NOTE: remove `-g` if upgrading from a local installation

```sh
# upgrade using npm
npm install -g @dendronhq/dendron-cli@latest

# upgrade using yarn
yarn upgrade @dendronhq/dendron-cli@latest
```
